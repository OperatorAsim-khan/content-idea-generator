# 💡 Content Idea Generator

An AI-powered web app that generates 10 unique content ideas instantly based on your niche. Built for solo creators who need fresh ideas fast.

---

## 🚀 Live Demo

> Open `index.html` in your browser — no installation needed.

---

## ✨ Features

- Enter any niche and get 10 ready-to-use content ideas
- Powered by Groq AI (LLaMA 3.3 model)
- Clean, modern dark UI
- Fully responsive on mobile
- Instant results with smooth animations

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Groq AI API (LLaMA 3.3 70B)

---

## ⚙️ Setup & Usage

1. Clone or download this repository
2. Get a free API key from [console.groq.com](https://console.groq.com)
3. Open `script.js` and replace `your_api_key_here` with your Groq API key
4. Open `index.html` in your browser
5. Type your niche and hit **Generate Ideas**
